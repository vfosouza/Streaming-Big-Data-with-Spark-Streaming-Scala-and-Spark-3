# Privacy Policy

## Introduction
This Privacy Policy describes how your data is collected, used, and shared when you interact with our Spark Streaming academic project.

## Data Collection
Our application collects public tweets from Twitter's API v2 Sample Stream. We collect:
- Tweet text content
- Tweet creation timestamps
- Author IDs

## How We Use Data
The data is used exclusively for:
- Academic research and educational purposes
- Data analysis and processing demonstrations
- Temporary storage for stream processing experiments

## Data Storage & Security
- Tweet data is temporarily stored as text files
- Files are named with timestamp identifiers ("Tweets_" + timestamp)
- Data collection is limited to 1000 tweets per session
- No personally identifiable information beyond what's available in public tweets is stored
- Data is stored locally and not shared with third parties

## Data Retention
Tweet data is retained only for the duration necessary for academic analysis and is not permanently stored or used for commercial purposes.

## Your Rights
As this is a non-commercial academic project using publicly available data, we do not track individual users. If you believe your Twitter content has been stored by our application and wish to have it removed, please contact us.

## Changes to This Policy
We may update this Privacy Policy from time to time to reflect changes in our practices or for other operational, legal, or regulatory reasons.
